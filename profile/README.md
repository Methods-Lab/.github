## Neural Network Visualization

> *See inside the black box. Understand, debug, and communicate neural network behaviour.*

<div align="center">

| Repository | Stack | What It Does |
|:---|:---:|:---|
| [**tensorspace**](https://github.com/Methods-Lab/tensorspace) | `JS` `Three.js` `WebGL` | 3D browser-based tensor flow renderer — watch activations propagate layer by layer in real time |
| [**netron**](https://github.com/Methods-Lab/netron) | `JS` `Electron` `ONNX` | Industry-standard model viewer for 30+ formats: ONNX, TF, PyTorch, CoreML, Keras |
| [**cnn-explainer**](https://github.com/Methods-Lab/cnn-explainer) | `JS` `Svelte` `D3` | Interactive CNN walkthrough — convolution, pooling, activation, all animated step by step |
| [**pytorch-cnn-visualizations**](https://github.com/Methods-Lab/pytorch-cnn-visualizations) | `Python` `PyTorch` | Grad-CAM, Guided Backprop, Saliency Maps, Smooth Grad — drop-in for any classifier |
| [**Tools-to-Design-or-Visualize-Architecture-of-Neural-Network**](https://github.com/Methods-Lab/Tools-to-Design-or-Visualize-Architecture-of-Neural-Network) | `Python` `LaTeX` | Curated toolkit: PlotNeuralNet, NN-SVG, Netscope, and 10+ others with usage examples |

</div>

```
  Rendering Pipeline:
  
  Model Weights ──► Layer Extraction ──► 3D Geometry ──► WebGL Render ──► Browser
                           │
                    ┌──────▼───────┐
                    │  Grad-CAM    │──► Heatmap Overlay
                    │  Saliency    │──► Pixel Attribution
                    │  Backprop    │──► Feature Visualization
                    └─────────────┘
```

## ML Infrastructure & Training

> *The engineering backbone — tools that make training, deploying, and managing models faster and reproducible.*

<div align="center">

| Repository | Role | Tech |
|:---|:---|:---:|
| [**traingenerator**](https://github.com/Methods-Lab/traingenerator) | Web UI → production-ready PyTorch/sklearn training scripts | `Python` `Streamlit` |
| [**inference**](https://github.com/Methods-Lab/inference) | High-performance vision model serving via REST API | `Python` `FastAPI` `Docker` |
| [**fvcore**](https://github.com/Methods-Lab/fvcore) | FLOPs analysis, parameter counts, PyTorch utilities (FAIR) | `Python` `PyTorch` |
| [**tangle**](https://github.com/Methods-Lab/tangle) | Lightweight experiment tracking and artifact versioning | `Python` |
| [**open_model_zoo**](https://github.com/Methods-Lab/open_model_zoo) | 200+ Intel OpenVINO-optimized models for edge deployment | `Python` `OpenVINO` |
| [**brpc**](https://github.com/Methods-Lab/brpc) | Baidu's industrial RPC for high-throughput ML serving | `C++` |

</div>

```
  Deployment Pipeline:

  Research Code                                          Production
  ─────────────                                          ──────────
  Experiment → [tangle] → Track
  Model     → [fvcore]  → Analyze FLOPs/Params
  Train     → [traingenerator] → Generate Scripts
  Weights   → [open_model_zoo] → Optimize for Hardware
  Serve     → [inference] → REST API
  Scale     → [brpc]      → High-throughput RPC
```

## Getting Started

<div align="center">

| Goal | Repository | First Step |
|:---|:---|:---|
| Visualize a neural network | [netron](https://github.com/Methods-Lab/netron) | `pip install netron` then `netron model.onnx` |
| Build a recommender system | [recommenders](https://github.com/Methods-Lab/recommenders) | `pip install recommenders` |
| Deploy a vision model | [inference](https://github.com/Methods-Lab/inference) | `pip install inference` |
| Understand CNNs visually | [cnn-explainer](https://github.com/Methods-Lab/cnn-explainer) | Open the live demo |
| Scale embeddings | [torchrec](https://github.com/Methods-Lab/torchrec) | `pip install torchrec` |
| Generate training code | [traingenerator](https://github.com/Methods-Lab/traingenerator) | Run the Streamlit app |
| Find pretrained models | [pretrained.ml](https://github.com/Methods-Lab/pretrained.ml) | Browse the model index |
