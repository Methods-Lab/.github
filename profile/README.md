## ЁЯза Neural Network Visualization

### ЁЯФ╖ [tensorspace](https://github.com/Methods-Lab/tensorspace)

`JavaScript` `Three.js` `TensorFlow.js` `WebGL`

---

### ЁЯФ╖ [netron](https://github.com/Methods-Lab/netron)
The industry-standard neural network model viewer. Supports ONNX, TensorFlow, PyTorch, Keras, CoreML, and 30+ other formats. Visualize architecture, weights, and metadata for any model.

`JavaScript` `Electron` `ONNX` `MIT License`

---

### ЁЯФ╖ [cnn-explainer](https://github.com/Methods-Lab/cnn-explainer)

`JavaScript` `Svelte` `D3.js` `MIT License`

---

### ЁЯФ╖ [pytorch-cnn-visualizations](https://github.com/Methods-Lab/pytorch-cnn-visualizations)
PyTorch implementations of CNN visualization techniques: Grad-CAM, Guided Backpropagation, Saliency Maps, Smooth Grad, and more. Drop-in ready for any PyTorch classification model.

`Python` `PyTorch` `Grad-CAM` `MIT License`

---

### ЁЯФ╖ [Tools-to-Design-or-Visualize-Architecture-of-Neural-Network](https://github.com/Methods-Lab/Tools-to-Design-or-Visualize-Architecture-of-Neural-Network)

## ЁЯОп Recommendation Systems

> From collaborative filtering to graph neural networks тАФ the full spectrum of modern recommender research.

### ЁЯФ╖ [recommenders](https://github.com/Methods-Lab/recommenders)
Microsoft's production-grade recommendation systems library. Implements best practices for building, evaluating, and deploying recommenders тАФ covers collaborative filtering, content-based, deep learning, and hybrid approaches.

`Python` `PyTorch` `TensorFlow` `MIT License`

---

### ЁЯФ╖ [torchrec](https://github.com/Methods-Lab/torchrec)
PyTorch domain library for large-scale recommendation systems. Handles sparse embeddings at Facebook/Meta scale тАФ distributed training, sharding strategies, and high-performance embedding lookup.

`Python` `PyTorch` `CUDA` `BSD-3 License`

---

### ЁЯФ╖ [monolith](https://github.com/Methods-Lab/monolith)
ByteDance's collision-free embedding table for recommendation systems. Enables real-time learning without ID collisions тАФ the architecture powering TikTok's recommendation engine.

`Python` `C++` `Embedding Tables` `Production ML`

---

### ЁЯФ╖ [RecFM](https://github.com/Methods-Lab/RecFM)
Factorization Machine-based recommendation framework. Implements FM, DeepFM, NFM, AFM, and xDeepFM with unified training and evaluation pipelines.

`Python` `Apache 2.0`

---

### ЁЯФ╖ [GNN-Recommender-Systems](https://github.com/Methods-Lab/GNN-Recommender-Systems)
Survey and implementation repository for graph neural network-based recommenders. Covers LightGCN, NGCF, PinSage, and social-aware GNN recommenders with benchmark datasets.

`Python` `PyTorch Geometric` `GNN` `Research`

---

### ЁЯФ╖ [Deep-Learning-for-Recommendation-Systems](https://github.com/Methods-Lab/Deep-Learning-for-Recommendation-Systems)
Comprehensive resource covering deep learning approaches in recommender systems тАФ Wide & Deep, DIN, DIEN, sequence modeling, and multi-task learning, with paper links and code.

`Python` `Deep Learning` `Research Survey`

---

### ЁЯФ╖ [Surprise](https://github.com/Methods-Lab/Surprise)

`Python` `scikit` `BSD-3 License`
